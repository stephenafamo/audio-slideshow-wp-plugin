Create a slideshow with audio.

== Description ==

With this plugin, you will be able to create a slideshow and add audio to it.
The slides will change according to the position of the audio file.
Useful for presentations, talks e.t.c.

Almost like video, but will take a lot less bandwith. So save your server costs

= Usage =

1. Go to `slideshows` in the menu and create a slideshow.
2. Near the tile, you will be shown a shortcode. Copy and paste it into the body of any post where you want the slideshow to be.
